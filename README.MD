# Student Performance Analysis

## 📌 Objective

This project analyzes how study hours and attendance affect student performance.

## 📊 Dataset

The dataset contains:

* StudyHours: Hours studied per day
* Attendance: Attendance percentage
* Marks: Student marks

## ⚙️ Steps Performed

1. Data Loading and Inspection
2. Data Cleaning (handled missing values & removed outliers)
3. Feature Engineering (Performance, EffortScore)
4. Data Analysis
5. Group-Based Analysis

## 📈 Insights

* Higher study hours lead to better marks.
* Attendance positively impacts performance.
* Students with high effort score perform better.
* Low attendance often results in poor marks.
* Excellent performers maintain both high study hours and attendance.

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```
